1. Apa gunanya IFRAME ?

    2. iFrame adalah singkatan dari Inline Frame, sebuah komponen dari file HTML yang memungkinkan Anda menyematkan video, dokumen, dan file lainnya ke dalam sebuah halaman website. Proses untuk menyematkan file dengan teknik ini disebut sebagai embed iFrame

  2. Apa beadnya FRAME dan IFRAME ?

    3. Frame dan Iframe pada dasarnya hampir sama. Bedanya, frame bisa dikelompokkan dengan frameset. Selain itu, ada atribut rows dan cols yang menentukan bagaimana susunan frame yang ditampilkan dan persentase ukuran masing-masing frame. Kode HTML di bawah ini hampir sama dengan yang di atas, dan perbedaannya hanya pada atribut cols yang saya ganti dengan rows.
